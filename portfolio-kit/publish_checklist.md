# 個人作品集上線清單

## 30 分鐘快速版

1. 建立 GitHub Profile repo（repo 名稱 = 你的 GitHub ID）。
2. 將 `profile_README_portfolio.md` 貼到 Profile repo 的 `README.md`。
3. 更新每個專案的 Description（用 `project_descriptions_for_github.md`）。
4. Pin 4~5 個精選專案到首頁。
5. 每個精選專案 README 至少補齊：目標、方法、結果、執行方式。

## 品質檢查（上線前）

1. 是否每個專案都能在 10 秒內看懂價值。
2. 是否至少 2 個專案有量化結果（例如報酬率、處理筆數、延遲）。
3. 是否所有組隊專案都清楚寫出「我的貢獻」。
4. 是否有 broken links 或看不到的圖片。
5. 是否避免把未完成作業或雜訊檔案放在精選 repo。
