# GitHub 個人作品集套件

這份套件是「非特定公司」版本，目標是把你的 GitHub 做成長期可維護的個人作品集。

## 內容

- `profile_README_portfolio.md`：GitHub 個人首頁可直接貼上的版型。
- `project_descriptions_for_github.md`：每個 repo 的一句話介紹（貼在 GitHub Description）。
- `publish_checklist.md`：快速上線清單。
- `project_case_study_template.md`：每個專案都可套用的 README 模板。
- `computational_cognitive_neuroscience_case.md`：你的計算認知神經科學專題（已整理成作品集敘事）。

## 建議精選專案（Pin 4~5 個）

1. `tw50_auto_trading_project`
2. `data_engineering_final_project`
3. `classroom_alert`
4. `web3_assignment`
5. `computational_cognitive_neuroscience_face-control`（建議新開 repo 放這份專題）

## 核心原則

1. 專案首頁 10 秒內要看懂：做什麼、怎麼做、結果如何。
2. 每個 repo 都要有可執行入口（命令、影片、或截圖流程）。
3. 組隊專案要明確標記你的貢獻，避免面試時責任邊界不清楚。
